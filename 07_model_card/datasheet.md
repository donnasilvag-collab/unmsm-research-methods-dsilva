# Ficha de datos prevista

## Naturaleza de los datos

El estudio prevé trabajar con un conjunto de datos mixto compuesto por:

- respuestas de encuesta sobre madurez de gestión de riesgos y controles de seguridad;
- registros documentales autorizados, como políticas, lineamientos o evidencias de trazabilidad;
- información cualitativa derivada de entrevistas semiestructuradas.

## Procedencia

Las fuentes provendrán de empresas peruanas de desarrollo de software que acepten participar en el estudio bajo consentimiento informado y resguardo de confidencialidad.

## Tratamiento esperado

- Anonimización de nombres de empresa, personas y proyectos.
- Separación entre insumos brutos, datos procesados y matrices analíticas.
- Documentación de variables, reglas de codificación y criterios de exclusión.

## Restricciones

No se publicarán datos sensibles, credenciales, fragmentos reales de código fuente ni evidencias que permitan identificar organizaciones participantes sin autorización expresa.
